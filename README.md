# Product Status Notifier

## Description
Scrappes product pages of specific sites and checks for price/availability changes. Sends notification using IFTTT webhooks if changed.

### Scrapers
    1. Newegg.com
    2. Tigerdirect.com
    3. B&H Photo
    4. SabrePC.com

### Setup Notifications
 - Uses IFTTT webhooks
